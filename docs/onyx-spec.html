Onyx Theory Specification.

Disclaimer: This page, and the rest of the Onyx Spec, may be out of date and/or incomplete. See this notice.

Onyx is an abandoned phonetic steno theory built around purity, consistency, and shortness.

Disclaimer: Onyx is an old theory that I no longer use, and no dictionary was ever made. I ended up moving to a fully orthographic theory, Pinchord. However, I think there are some cool ideas here if anyone is interested in them.

Bear with me as I try to document something I haven't used in months :D

Outline

Onyx is a (hopefully) accent-independent minimal phonetic steno theory for the WSI/Extended Stenotype layouts.

  • Syllabic Splitting
  • Mandatories
  • Conflict resolution via orthography, *, etc.
