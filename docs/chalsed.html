Chalsed

Pronounced KAL-SED (/kælsɛd/)

The Lisp Editor.

Just something I've been thinking about for a while.

I'm a vimmer, but I have my grievances, especially when it comes to editing Lisp.

Vim is pretty antiquated, and I don't like the emphasis on plugins in Neovim.

I imagine an editor that isn't text based but code based (or rather collection based, but those are the same thing in Lisp). This is my real problem with every editor I've ever used, they're all text editors through and through, and there's no amount of extension that can change that.

I figure it's good to have at least one project that you've been thinking about writing for months and never actually do, so here that is :D
